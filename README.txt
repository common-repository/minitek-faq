== Description ==

Minitek FAQ is an Ajax-driven, mobile-friendly, feature-rich FAQ (Frequently Asked Questions) plugin for WordPress.

It features an innovative topic navigation menu which can be used to effortlessly add multiple structured FAQ sections to your website.

= Info =

Visit our [official product page](https://www.minitek.gr/wordpress/plugins/minitek-faq) to see detailed info on the plugin.

For a live demo of the free version of the plugin, go to [http://demo.minitek.gr/#wp_minitek_faq](http://demo.minitek.gr/#wp_minitek_faq).

For more advanced features (eg. User questions, voting system), you can upgrade to the [Premium version](https://www.minitek.gr/wordpress/plugins/minitek-faq).

For a live demo of the Premium version of the plugin, go to [http://demo.minitek.gr/#wp_minitek_faq_pro](http://demo.minitek.gr/#wp_minitek_faq_pro).
